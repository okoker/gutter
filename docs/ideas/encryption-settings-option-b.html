Danger zone

Destructive operations. Each requires explicit confirmation in a separate modal.

Remove all encryption from this workspace
Decrypts every .emd file in the workspace back to plain .md. Files become readable by any text editor or file viewer.
⚠️ This is irreversible by the app. To re-encrypt, you'd have to encrypt each file again. Files will be readable on disk and any sync target (iCloud, Dropbox, GitHub, etc.) immediately after this completes.
Delete all saved passphrases
Removes the named passphrase address book. Encrypted files stay encrypted; you'll have to re-enter passphrases manually to open them.

Help & reference

Plain-language explanations of what's actually happening under the hood.

  • How encryption works in this app
    A single page covering the file format, key derivation, and what's protected vs not.
  • What happens if I forget my passphrase?
    Honest answer: the file is gone. There is no recovery. Plan accordingly.
  • Sharing encrypted files with others
    In Phase 1, share by giving the recipient the passphrase out-of-band. Multi-recipient comes in Phase 2.
  • What is Diceware?
    Random word selection from a 7,776-word list. 6 words ≈ 77 bits entropy — far stronger than typical passwords, easier to write down.
  • Algorithms used
    XChaCha20-Poly1305 + Argon2id + zstd. Parameters per file, future-proof.
