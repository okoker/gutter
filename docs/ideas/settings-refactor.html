Danger zone

Danger zone

Destructive operations. Each requires explicit confirmation.

Remove all encryption from this workspace
Decrypts every .emd file back to plain .md. Files become readable by any text editor.
⚠️ Irreversible by the app. Files will be readable on disk and any sync target (iCloud, Dropbox, GitHub) immediately after this completes.
Delete all saved passphrases
Removes the named passphrase address book. Encrypted files stay encrypted; you'll need to re-enter passphrases manually.
